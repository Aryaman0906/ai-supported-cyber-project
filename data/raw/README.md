# Raw synthetic data

This folder contains safe synthetic/generic starter examples only.

Rules:

- Do not commit real Gmail exports, private inbox data, OAuth tokens, API keys, generated reports, or personal data.
- Do not commit live phishing URLs or active malicious infrastructure.
- Public datasets should not be added until licensing, source quality, privacy, and safety have been reviewed.
- Use reserved/example domains and test-only wording for starter data.

Files in this PR:

- `synthetic_text_examples.csv` — safe synthetic email/text examples.
- `synthetic_url_examples.csv` — safe synthetic/reserved URL examples.
